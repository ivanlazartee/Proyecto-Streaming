## 🎬 OnDev+

Plataforma de streaming de películas y series desarrollada como proyecto frontend utilizando tecnologías web modernas.
El proyecto cuenta con una interfaz responsive, diseño dinámico e interacción visual orientada a mejorar la experiencia del usuario.

## 🚀 Tecnologías Utilizadas
- HTML5
- CSS3
- Bootstrap
- CSS Grid
- Flexbox

## ✨ Funcionalidades
- 📱 Diseño completamente responsive
- 🎞️ Carruseles interactivos
- 🎬 Categorías dinámicas de películas y series
- 🔍 Buscador de contenido
- 🔐 Sistema de login visual
- 🎨 Interfaz moderna inspirada en plataformas de streaming
